Access to the Alphabet type.
Alphabet<T>::Type
Include Headers
seqan/graph_types.h
Parameters
T
Type T must be a type that uses some kind of alphabet internally.
Types: Graph
Remarks
Type T can be for example an automaton where the alphabet type describes the domain of the transition labels.
Return Values

Alphabet type.
