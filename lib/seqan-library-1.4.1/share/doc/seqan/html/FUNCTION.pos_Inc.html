Function
posInc
Increments the local offset of a position type.
posInc(pos)
Include Headers
seqan/sequence.h
Parameters
pos
A position type. Could either be an integer seqOfs or a pair (seqNo, seqOfs). In both cases seqOfs will be incremented by one.
