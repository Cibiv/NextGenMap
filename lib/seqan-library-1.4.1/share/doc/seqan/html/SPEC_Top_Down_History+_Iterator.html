TopDownHistory Iterator
String tree iterator that can go down, right, and up. Supports depth-first search.

Remarks
If not copy-constructed the TopDownHistory Iterator starts in the root node of the string tree. Depending on the depth-first search mode the root is not the first DFS node. To go to the first DFS node use goBegin.
Note that the virtual tree can either be a virtual suffix tree or a virtual prefix tree. The suffix tree is shown in Figure 1 and is implemented with the IndexDfi, IndexEsa and IndexWotd. In contrast the FMIndex implements a prefix trie shown in Figure 2.
