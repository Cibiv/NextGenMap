Function
length
The number of characters in the underlying text of the index is returned.
length(index)
Include Headers
seqan/index.h
Parameters
index
The index to return the number of characters of.
Types: Index
Remarks
If the underlying text is a StringSet then the sum of all characters of the sequneces in the string set is returned.
Return Values
The number of characters in the raw underlying text of the index is returned.
