Tag
Prefix Order
Specify whether less-than or greather-than comparison is meant.
Include Headers
seqan/sequence.h
Remarks
The default for all comparison functions is TagPrefixLess.
Tags
TagPrefixLess
A prefix is smaller.
For example: "abc" < "abcde".
TagPrefixGreater
A prefix is greater.
For example: "abc" > "abcde".
