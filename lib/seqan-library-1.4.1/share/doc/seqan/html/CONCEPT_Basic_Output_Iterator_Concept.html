Examples
In the following, x is an iterator to type X, t is a valid rvalue of type X.
The following expressions must be valid.
*x = t     // Dereference assignment.
++x        // Preincrement.
(void)x++  // Postincrement.
*x++ = t   // Postincrement and assign.
 
assignValue(x, t);
