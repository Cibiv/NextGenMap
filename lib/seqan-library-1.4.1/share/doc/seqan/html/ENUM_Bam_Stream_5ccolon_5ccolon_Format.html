Enum
BamStream::Format
Select the format to use for reading/writing.
Include Headers
seqan/bam_io.h
Values
AUTO
Auto-detect format from file content on reading and from the file name on writing. If Auto-detection fails, SAM is used.
SAM
Force reading/writing of SAM.
BAM
Force reading/writing of SAM.
