Macro
SEQAN_VERIFY_CHECKPOINTS
Verify check points for the given file name.
SEQAN_VERIFY_CHECKPOINTS(path)
Parameters
path
Path to the file to verify check points for. Relative to parent directory of tests.
Examples
SEQAN_VERIFY_CHECKPOINTS("core/include/seqan/basic_alphabet.h");
