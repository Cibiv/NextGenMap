Function
waitFor
Waits for an asynchronous request to complete.
waitFor(request[, timeoutMilliSec])
Include Headers
seqan/file.h
Parameters
request
Reference to an AsyncRequest object.
Types: AsyncRequest
timeoutMilliSec
Timout value in milliseconds.
Default: Infinity.
Remarks: A value of 0 can be used to test for completion without waiting.
Remarks
waitFor suspends the calling process until request is completed or after timeoutMilliSec milliseconds.
Return Values
A bool which is true on completion and false on timeout.
