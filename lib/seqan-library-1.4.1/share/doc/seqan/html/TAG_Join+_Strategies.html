Join Strategies
Tags used for selecting journaling strategies when joining a Journaled String to a global reference sequence.
Include Headers
seqan/align.h
Tags
JournaledManhatten
Constructs Journaled String based on Manhatten distance.
Remarks: This strategy is very fast on the cost of memory.
JournaledCompact
Computes an optimal alignment to construct Journaled String.
Remarks: This strategy is slow but depending on the scoring function minimizes the memory requirements for the computed Journaled String (default).
