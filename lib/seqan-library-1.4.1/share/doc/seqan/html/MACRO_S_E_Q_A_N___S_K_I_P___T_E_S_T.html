Macro
SEQAN_SKIP_TEST
Force the test to return without failing and mark it as skipped.
SEQAN_SKIP_TEST
Examples
SEQAN_DEFINE_TEST(test_skipped)
{
    SEQAN_SKIP_TEST;
}
