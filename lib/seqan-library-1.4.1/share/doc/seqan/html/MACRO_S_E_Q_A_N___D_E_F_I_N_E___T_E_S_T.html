Macro
SEQAN_DEFINE_TEST
Expand to test definition.
SEQAN_DEFINE_TEST(test_name)
Parameters
test_name
The name of the test.
Remarks
This macro expands to the definition of a void function with SEQAN_TEST_ + test_name as its name.
Examples
SEQAN_DEFINE_TEST(test_name)
{
   SEQAN_ASSERT_LT(0, 3);
}
